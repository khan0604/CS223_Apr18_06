# Grade Management System



A software for management of students grades. This software can be accessed by

  - Students
  - Faculties
  

## Students

Student can select any options from the list :

  - View Profile
  - Modify Profile
  - Add Courses
  - View Courses
  - View Stats

## Faculties

Faculties can select any options from the list :

  - View Profile
  - Modify Profile
  - Add Courses
  - View Courses
  - View All Stats
  - Update Marks
  - Modify Grades
